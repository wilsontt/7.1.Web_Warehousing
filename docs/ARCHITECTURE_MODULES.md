# 模組架構（8 大功能）

**建立日期**: 2025/11/15  
**最後更新**: 2025/11/15

本文件定義系統 8 大功能的目錄與對應之前端/文件位置，作為後續開發與維護之依據。

## 目錄與對應

| 主功能 | 目錄 | 主要內容 | 既有前端路由 | 文件位置 |
|---|---|---|---|---|
| 1. 基本作業 | `modules/1.基本作業/` | 基礎資料維護 | `/basic-operations/code/basic`（代碼維護） | `modules/1.基本作業/1.1.代碼維護/docs/` |
| 2. 收發作業 | `modules/2.收發作業/` | 收貨/發貨/退貨/點收 | - | - |
| 3. 調撥作業 | `modules/3.調撥作業/` | 倉內/倉間調撥、移位 | - | - |
| 4. 報表查詢 | `modules/4.報表查詢/` | 統計/明細報表、查詢 | - | - |
| 5. 系統設定 | `modules/5.系統設定/` | 參數/流程/環境設定 | - | - |
| 6. 安全管理 | `modules/6.安全管理/` | 帳號/角色/權限/稽核/2FA | - | - |
| 7. 客戶管理 | `modules/7.客戶管理/` | 客戶主檔、合約、計費 | - | - |
| 8. 系統維護 | `modules/8.系統維護/` | 排程、修復、工具 | - | - |

## 原則
- 架構以「主功能 → 子功能」樹狀目錄組織。
- 既有模組逐步對齊後，方才考慮實體搬移；在此之前以 README 索引對應。
- 新增子功能時，請於對應主功能下建立 `X.Y.子功能名稱/` 與 README。
