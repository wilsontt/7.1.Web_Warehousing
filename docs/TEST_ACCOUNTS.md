# 測試帳號 (Test Accounts)

**建立日期**: 2025/11/15  
**最後更新**: 2025/11/15

## 測試帳號清單

本文件提供用於測試系統的測試帳號資訊。所有測試帳號僅供開發與測試環境使用，**不得用於正式環境**。

### 管理員帳號 (Admin)

| 項目 | 值 |
|------|-----|
| 帳號 | `admin` |
| 密碼 | `Admin@123` |
| 角色 | 管理員 (admin) |
| 權限 | 所有權限 (`*`) |
| 說明 | 擁有系統所有功能的使用權限，可用於測試系統管理作業 |

### 主管帳號 (Manager)

| 項目 | 值 |
|------|-----|
| 帳號 | `manager` |
| 密碼 | `Manager@123` |
| 角色 | 主管 (manager) |
| 權限 | 基本作業、例行作業、庫存作業、客戶作業、移儲作業、稽核作業、倉儲管理作業 |
| 說明 | 擁有大部分業務功能的使用權限，但不包含系統管理作業 |

### 一般使用者帳號 (User)

| 項目 | 值 |
|------|-----|
| 帳號 | `user` |
| 密碼 | `User@123` |
| 角色 | 一般使用者 (user) |
| 權限 | 基本作業、例行作業、庫存作業（僅查詢） |
| 說明 | 擁有基本查詢與作業功能的使用權限，用於測試一般使用者操作流程 |

## 使用注意事項

1. **僅供測試使用**：這些帳號僅供開發與測試環境使用，請勿在正式環境中使用。
2. **密碼強度**：測試帳號密碼符合系統密碼政策要求（至少 8 字元，包含大小寫字母、數字和特殊字元）。
3. **權限測試**：使用不同角色帳號登入，可測試系統的權限控制功能。
4. **資料安全**：測試帳號的操作會產生測試資料，請定期清理測試環境。

## 測試場景建議

### 管理員測試場景
- 登入系統管理作業模組
- 測試使用者帳號角色設定
- 測試系統參數設定
- 測試權限控管設定

### 主管測試場景
- 登入各種業務作業模組
- 測試單據審核功能
- 測試報表查詢功能
- 驗證無法存取系統管理作業

### 一般使用者測試場景
- 登入基本作業模組
- 測試資料查詢功能
- 測試基本資料維護功能
- 驗證權限限制（無法存取管理功能）

## 密碼變更

如需變更測試帳號密碼，請聯繫系統管理員。變更後請更新本文件。
