# 系統 8 大功能模組總覽

**建立日期**: 2025/11/15  
**最後更新**: 2025/11/15

此目錄描述系統之 8 大功能的樹狀結構與對應的程式/文件位置。實際程式碼可逐步搬移或以連結方式對應至既有模組。

## 目錄結構

- `1.基本作業/`
  - `1.1.代碼維護/` → 前端：`/basic-operations/code/basic`；文件：`modules/1.基本作業/1.1.代碼維護/docs/`
- `2.收發作業/`
- `3.調撥作業/`
- `4.報表查詢/`
- `5.系統設定/`
- `6.安全管理/`
- `7.客戶管理/`
- `8.系統維護/`

## 命名規範
- 主功能以數字排序：`1.` ~ `8.`
- 子功能以「主功能編號.序號.子功能名稱」方式命名，例如：`1.1.代碼維護/`

## 既有模組對應
- 基本作業 → 代碼維護
  - 前端路由：`/basic-operations/code/basic`
  - 文件目錄：`modules/1.基本作業/1.1.代碼維護/docs/`

後續新子功能一律置於對應主功能目錄下，並建立 README 與對應實作位置。
